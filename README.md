## Basic overview
This platform is a web-based decision-support tool designed to make Life Cycle Assessment (LCA) simple, automated, and circularity-focused for the metals sector (e.g., aluminium, copper, critical minerals). Unlike traditional LCA tools that require deep expertise, this platform leverages AI/ML models and LCI databases to generate actionable sustainability insights — helping engineers, metallurgists, and decision-makers evaluate both environmental impacts and circularity potential across the full metal value chain.
